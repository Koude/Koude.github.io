# 路由器设置
进入系统管理 - 系统设置 - SSH 连接，启用SSH连接。

# 设置hosts
```ssh admin@192.168.50.1
cd /jffs```
新建文件 `dnsmasq.conf.add` 并写入内容
```addn-hosts=/jffs/configs/hosts
vi /jffs/configs/hosts```
写入内容
```192.168.0.1 www.example.com
```

# 重启 DNS 服务
```service restart_dnsmasq```
